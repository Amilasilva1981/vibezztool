Why it's worth using

Reduce your risk of getting hacked

Short or reused passwords are the single most common way social media and email accounts get compromised.

Passwords never leave your device

This generator uses your browser's built-in secure random number generator (crypto.getRandomValues) — nothing is sent to a server or logged anywhere.

Built for real accounts

You can customize length and character types to match specific platform requirements, from simple PINs to complex 32-character passwords.

How to use it

  1. Set your preferred password length using the slider.
  2. Choose which character types to include — uppercase, lowercase, numbers, and symbols.
  3. Optionally exclude ambiguous characters like l, 1, O, and 0 if you'll be typing the password by hand.
  4. Click "Generate," then "Copy," and paste it into your account's password field.

Pro tip

Use a different, generated password for every account. If that's hard to remember, pair this tool with a password manager so you only need to remember one master password.
